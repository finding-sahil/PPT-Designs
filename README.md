# PPT Designs

A collection of premium PowerPoint-inspired web designs and presentations.

## Project Structure

This repository contains several web-based presentation designs, each in its own directory:

- **[Adv PPT](./Adv PPT)**: An advanced presentation design with interactive elements and teambuilding data.
- **[PPT 1](./PPT 1)**: First iteration of the presentation design.
- **[PPT 2](./PPT 2)**: Second iteration with layout improvements.
- **[PPT 3](./PPT 3)**: Third iteration with refined styling.
- **[PPT 4](./PPT 4)**: Fourth iteration focusing on responsiveness.
- **[PPT 5](./PPT 5)**: Fifth iteration with additional components.

## How to View

To view any of the designs, simply open the `index.html` file within the respective directory in your web browser.
